# 家計簿アプリ 使い方ガイド

スマホで使える家族共有の家計簿アプリです。レシートを撮影するだけで自動入力でき、資産管理や将来のシミュレーションもできます。

---

## 基本的な使い方

### ログイン

アプリを開いたら「Google でログイン」をタップし、Google アカウントで認証します。

---

## 1. レシートを記録する

### カメラで撮影（自動入力）

1. ホーム画面で **「📷 カメラで撮影」** をタップ
2. レシートを撮影
3. **「🔍 OCR で読み取る」** をタップ
4. 読み取り結果を確認・修正して **「💾 Google Sheets に保存」**

> 複数枚まとめて読み取ることもできます（「🖼 ライブラリから選択」で複数選択）。

### 手動で入力

1. ホーム画面で **「✏️ 手動で入力」** をタップ
2. 日付・カテゴリ・支払先・金額・使用者を入力
3. **「💾 Google Sheets に保存」**

### 自動保存モード

「自動保存モード」をオンにすると、OCR 読み取り後に確認画面を省略して自動保存します。

---

## 2. 家計を集計する（📈 ボタン）

ホーム右上の **📈** をタップします。

### 月次タブ
- 当月の支出を円グラフで表示
- **‹ ›** で前後の月に移動
- 予算が設定されているカテゴリは予算ラインと比較表示

### 年次タブ
- 月別支出の棒グラフ（年間推移）
- 年間のカテゴリ別円グラフ
- **‹ ›** で前後の年に移動

---

## 3. 資産管理（💰 ボタン）

ホーム右上の **💰** をタップします。

### 残高入力タブ

毎月の各口座残高を入力します。

| 項目 | 内容 |
|---|---|
| 銀行預金 | ゆうちょ・楽天銀行・千葉銀行など |
| 保険 | 終身保険・個人年金など |
| 株式資産 | 証券口座の残高 |
| 貴金属資産 | 金・貴金属の残高 |
| 年金残高 | 確定拠出年金・退職金見込み |
| 負債 | 自動車ローン残高など |

入力後、**「💾 保存する」** をタップ。

**前月の残高を引き継ぐ**（当月データなしの場合に表示）：前月の値をそのまま引き継いで入力欄に設定します。変更があった項目だけ修正してから保存できます。

### 収支タブ

月次の収入・支出を入力します。

- **家計（家計簿より）**: 家計簿の支出合計が自動で表示されます（編集不可）
- その他の項目は手動入力
- **前月ボタン**（各項目の右に表示）：タップすると前月の金額を自動入力できます

### 推移グラフタブ

全期間の資産推移をグラフで確認できます（カテゴリ別積み上げ＋純資産ライン）。

---

## 4. 収支予測（📅 ボタン）

資産管理画面の **📅** をタップします。

年単位で毎月の収入・支出の予測を入力します。

| 入力項目 | 収入 | 支出 |
|---|---|---|
| 対象年 | 上部の「対象年」で選択 | 同左 |
| 実績 | 実際の月額 | 実際の月額 |
| 予測 | 見込みの月額 | 見込みの月額 |

入力後、**「保存する」** をタップ。

### ライフイベントの登録

下部のフォームで一時的な大型支出を登録できます。

- 年月・イベント名・金額・メモを入力して **「＋ イベントを追加」**
- 登録済みイベントは一覧表示され、**「削除」** で削除できます

例：大学入学金・車の買い替え・リフォームなど

---

## 5. シミュレーション（🧮 ボタン）

資産管理画面の **🧮** をタップします。

直近の資産残高を起点に、収支予測とライフイベントを反映した将来20年間の資産推移グラフを表示します。

- 収支予測・ライフイベントを登録してから確認すると正確な予測になります
- ライフイベントは縦線マーカーで表示

---

## 移動中に保存し忘れた場合

資産管理・収支画面で入力後に保存せず別の画面に移動しようとすると、確認ダイアログが表示されます。「このまま移動する」を選ぶと入力内容が失われます。

---

## スプレッドシートを直接確認する

ホーム右上の **📊** をタップするとGoogle Sheetsが開きます。

### シートの構成

| タブ名 | 内容 |
|---|---|
| `2026.1`〜`2026.12` | 月別の家計簿データ |
| `資産管理` | 月次の資産・負債残高 |
| `キャッシュフロー` | 月次の収入・支出 |
| `収支予測` | 年次の収支予測データ |
| `ライフイベント` | 登録済みのライフイベント |

---

## その他の操作

| 操作 | 方法 |
|---|---|
| ログアウト | ホーム右上の **⚙** → ログアウト |
| OCRモデル切替 | ホーム画面の「モデル」セレクト（Haiku：速い / Sonnet：精度高め） |
| 解像度切替 | ホーム画面の「解像度」セレクト（高いほど精度が上がるが遅くなる） |
